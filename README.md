# 📊 Financial Intelligence Dashboard (Em Desenvolvimento)

> **Unindo 25 anos de expertise em Controladoria com a potência do Desenvolvimento Full Stack.**

Este projeto nasce da necessidade real de gestores financeiros: a capacidade de não apenas olhar para o passado, mas simular o futuro com precisão matemática. É uma ferramenta de Business Intelligence focada em fluxo de caixa estratégico.

---

## 🎯 O Problema
A maioria das ferramentas de controle financeiro são "estáticas" — elas mostram o que aconteceu, mas falham em ajudar o gestor a tomar decisões preventivas diante de mudanças de mercado.

## 💡 A Solução
Um dashboard interativo que permite realizar **Análises de Sensibilidade (What-if Analysis)** em tempo real. O sistema utiliza algoritmos de projeção para recalcular o futuro financeiro da empresa com base em variáveis dinâmicas (inflação, variação cambial, queda de demanda).

---

## 🛠️ Roadmap de Desenvolvimento (Plano de Voo)

### Fase 1: Arquitetura e Estrutura (Current)
- [ ] Modelagem do banco de dados relacional (PostgreSQL) focado em séries temporais.
- [ ] Estruturação do Backend (Node.js/TypeScript) com lógica de cálculo de projeção composta.
- [ ] Configuração do sistema de autenticação segura para dados sensíveis.

### Fase 2: Inteligência e Algoritmos
- [ ] Implementação da função `calculateProjection` para processamento de cenários Otimistas/Pessimistas.
- [ ] Criação de API Endpoints para alimentação de dados via CSV/JSON.

### Fase 3: Frontend & Experiência do Usuário (UI/UX)
- [ ] Desenvolvimento de gráficos dinâmicos com **Recharts**.
- [ ] Implementação de Sliders de Sensibilidade para simulação instantânea de variáveis.
- [ ] Dashboard responsivo com foco em clareza de indicadores (Burn rate, Runway, Cash Gap).

---

## 🧬 Diferenciais Técnicos
- **Lógica Matemática:** Cálculos de juros compostos e projeções lineares integrados diretamente na engine do sistema.
- **Visão de Negócio:** Aplicação de conceitos reais de controladoria (DRE, Fluxo Direto/Indireto) transformados em código.
- **Performance:** Otimização de queries para processamento de grandes volumes de transações.
